# 🧮 Inventory Summary Tool

This is a command-line tool that helps you manage a basic inventory of products.

## 📦 What It Does

- Add new products or update existing ones
- View all inventory with total value
- Highlights products with low stock (less than 5)
- Record product sales and update stock
- Saves everything to CSV files

## 🛠 How to Run It

1. Make sure Python is installed
2. Clone this repo and run:

```bash
python3 inventory_summary_tool.py
